<h1 align="center">
System Linked 
</h1>

##### A distributed Linux system monitoring platform built to track resource usage, processes, network activity, services, and logs across multiple systems in a customizable format. The platform uses lightweight Resource Daemons to collect data from kernel interfaces like /proc and /sys while keeping system overhead low. A central parser and analysis engine processes and normalizes the collected data, calculates metrics, and stores system snapshots for monitoring and troubleshooting. The platform provides detailed real-time analysis and remote monitoring through a centralized API and data streaming interface for easier system management.
